    CSS

    1. CSS는 HTML의 불편함에서 시작한 언어이다.
    2. CSS의 기본 문법
    3. STYLE태그와 STYLE 속성의 차이

    CSS는 HTML의 불편함에서 시작한 언어이다. HTML이라는 전자문서를 드러낼 수 있는 툴은 시간이 흘러감에 따라 사람들은 더 많은 욕구를 갖게 되었다. 이때 다양한 표현을 추가하고자 HTML에 다양한 태그가 등장하게 된다. 하지만 문제점이 존재한다. HTML은 그 자체로서 '정보'를 나타내야 하는데, 시각적 효과를 추가함으로써 HTML 본연의 정보라는 의미가 희석되는 단점이 생겼다. 또한 수백, 수천, 수만개의 코드를 하나 하나 수정해야한다면 그것만큼 고역도 없을 것이다. CSS는 거기에서 시작되었다.

    CSS란 새로운 언어를 컴퓨터에게 인식시키기 위해서 우리는 내가 지금 입력하는 내용은 HTML이 아닌 CSS로 읽어달라는 선언( declaration)을 해야하게 된다. 이것이 바로 (style)이라는 태그이다. style 태그를 사용한 뒤 내가 일괄적으로 변경하고 싶은 태그를 사용하면 (예를 들면 여기서는 a태그) 단 한 번의 입력으로 동일한 태그 모두를 수정할 수 있게 된다. 이때 선택된 a태그를 선택자(selctor)라고 한다. 또한 이때 a태그를 변경하는 내용을 효과(property)라고 부른다.

    style태그는 html에서 css언어로 해석하라고 나타낼 수 있는 장치이다. 반면 stlye 속성은 'X'라는 태그를 CSS 명령과 별도로 지정할 수 있는 태그이다. 단순히 한 가지만을 두고 둘을 비교하면 겉으로는 큰 차이가 없어보이나 결과적으로 style 태그는 같은 종류의 태그를 모두 변경할 수 있지만, style 속성은 단발성, 일회성이라는 성격을 지니게 된다.
